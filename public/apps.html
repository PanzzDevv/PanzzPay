Contoh Kode Integrasi REST API

Dukungan mudah untuk berbagai stack pemrograman backend.

Node.js / Express

const res = await fetch('http://localhost:3000/api/qris/generate', {
  method: 'POST',
  headers: { 'X-API-KEY': 'pz_live_your_key' },
  body: JSON.stringify({ base_amount: 15000 })
});

PHP / Curl

$ch = curl_init('http://localhost:3000/api/qris/generate');
curl_setopt($ch, CURLOPT_HTTPHEADER, ['X-API-KEY: pz_live_your_key']);
curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode(['base_amount' => 15000]));
$res = curl_exec($ch);

Python / Requests

import requests
res = requests.post('http://localhost:3000/api/qris/generate',
  headers={'X-API-KEY': 'pz_live_your_key'},
  json={'base_amount': 15000}
)
