            API DOCUMENTATION FOR DEVELOPERS
            

              
              Dokumentasi Integrasi PanzzPay REST API
            

            
Gunakan endpoint di bawah ini untuk menghubungkan PanzzPay langsung ke backend website/aplikasi Anda sendiri.

          


          

            
            
            

              

                POST
                /api/qris/generate
              

              

                Membuat QRIS Dinamis lengkap dengan total nominal tagihan dan kode unik.
              

              

                
curl -X POST \
  https://panzzpay.vercel.app/api/qris/generate \
  -H "Content-Type: application/json" \
  -H "X-API-KEY: pz_live_xxx" \
  -d '{"base_amount": 10000, "auto_unique": true}'

              

            


            
            

              

                POST
                /api/webhook/callback?token=pz_wh_xxx
              

              

                Menerima lemparan notifikasi HP dan mencocokkan status invoice ke PAID.
              

              

                
curl -X POST \
  "https://panzzpay.vercel.app/api/webhook/callback?token=pz_wh_xxx" \
  -H "Content-Type: application/json" \
  -d '{"message": "Pembayaran masuk Rp 10.338"}'

              

            


            
            

              

                GET
                /api/invoices/:id
              

              

                Mengecek status terkini suatu invoice (PENDING, PAID, atau EXPIRED).
              

              

                
curl -X GET \
  https://panzzpay.vercel.app/api/invoices/INV-xxx
